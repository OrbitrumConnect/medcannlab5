## 🎯 LINK PRINCIPAL

### **⚠️ IMPORTANTE: Verificar URL no Vercel**

O link `medcanlab1.0.vercel.app` pode não estar correto ou o projeto pode não estar deployado.

### **Como encontrar a URL correta:**

1. **Acesse o Vercel Dashboard:**
   - https://vercel.com/dashboard
   - Selecione seu projeto

2. **A URL estará no topo da página:**
   - Formato: `https://nome-do-projeto.vercel.app`
   - OU seu domínio personalizado se configurado

3. **Se o projeto não estiver deployado:**
   - Veja o guia: `GUIA_DEPLOY_VERCEL.md`
   - Faça o deploy primeiro

### **Link de Produção (após deploy):**
```
https://SEU-PROJETO.vercel.app
```

**Exemplos possíveis:**
- `https://medcanlab-3-0.vercel.app`
- `https://medcanlab3-0.vercel.app`
- `https://medcanlab1-0.vercel.app`
- OU seu domínio personalizado

## 📲 COMO FUNCIONA

Quando o paciente abre o link no celular ou computador:

1. **O navegador detecta que é um PWA instalável**
2. **Aparece opção para instalar:**
   - **Android:** "Adicionar à Tela Inicial"
   - **iOS:** "Adicionar à Tela de Início"
   - **Desktop:** Ícone de instalação na barra de endereço
3. **Paciente clica e o app é instalado**
4. **App aparece na tela inicial como aplicativo nativo**
