## 💳 SOBRE STRIPE / MERCADO PAGO

### Precisa ser agora para finalizar?
**Não necessariamente.** A integração de pagamento é a última peça, mas pode ser lançada em fases:

**Fase 1 (sem Stripe — lançável):**
- App funciona com `payment_status = 'trial'` (3 dias grátis calculado client-side)
- Profissionais já estão `exempt`
- Pacientes podem usar o app normalmente durante o trial
- Score: **~85%** sem Stripe

**Fase 2 (com Stripe — monetização):**
- Stripe Checkout real com 3 planos (R$150, R$250, R$350)
- Webhook atualiza `payment_status` automaticamente
- Score: **~95%** com Stripe

### O que é necessário para integrar Stripe:
1. Conta Stripe ativa (pode ser em modo Teste inicialmente)
2. `STRIPE_SECRET_KEY` e `STRIPE_WEBHOOK_SECRET`
3. Criar 3 produtos/preços no dashboard do Stripe
4. ~3 horas de implementação (2 Edge Functions + refactor `PaymentCheckout.tsx`)

### Alternativa: Mercado Pago
- Também possível, mas requer SDK diferente
- PIX nativo, boleto, cartão de crédito
- Mais comum no Brasil para pagamentos recorrentes
- Complexidade similar (~3h)

**Recomendação:** Avançar com Stripe em modo Teste para testar o fluxo completo, depois ativar produção quando estiver pronto para cobrar.
