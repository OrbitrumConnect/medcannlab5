# Memórias persistentes — espelho no repo

Espelho das **memórias persistentes mais recentes** do Claude Code, copiadas de
`~/.claude/projects/.../memory/` pra viajarem via git entre **desktop e laptop**
(as duas máquinas têm paths de `~/.claude` diferentes — ver CLAUDE.md, seção
"Diretório de memórias persistentes").

## O que é

- `MEMORY.md` — o índice (hierarquia, nível 1 = ler primeiro).
- Os demais `.md` — as ~25 memórias mais recentes (cobertura ~19/05 → 22/05).

## Fonte canônica

A fonte de verdade continua sendo o `~/.claude` de **cada máquina**. Esta pasta é
um **snapshot** pra sincronização cross-machine e backup. Quando trocar de
máquina: copiar destes arquivos pro `~/.claude/.../memory/` local, ou só ler
direto daqui.

Re-sincronizar = recopiar as mais recentes quando quiser. Total real de memórias
no `~/.claude`: 221 arquivos (22/05/2026) — aqui está só a ponta recente.
