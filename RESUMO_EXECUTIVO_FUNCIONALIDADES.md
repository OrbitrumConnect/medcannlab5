## 🤖 IA RESIDENTE - NÔA ESPERANÇA

### ✨ Funcionalidades Principais

| Funcionalidade | Descrição |
|---------------|-----------|
| **Chat Multimodal** | Texto + Voz (entrada e saída) |
| **Avaliação IMRE** | Protocolo pausado, uma pergunta por vez |
| **Análise Triaxial** | 3 fases da anamnese preservadas |
| **Multirracional** | 5 racionalidades médicas integradas |
| **Base de Conhecimento** | Acesso a documentos científicos |
| **Relatórios Automáticos** | Geração e salvamento automático |

### 🎯 Protocolo IMRE Triaxial

1. **I - Investigação**: Coleta de dados primários (fala espontânea preservada)
2. **M - Metodologia**: 
   - Abertura Exponencial
   - Desenvolvimento Indiciário
   - Fechamento Consensual
3. **R - Resultado**: Análise multirracional (5 racionalidades)
4. **E - Evolução**: Recomendações e plano de acompanhamento
